# 10476 THUNDERBOTS ⚡

Codebase for FIRST Robotics Competition 2026 Rebuilt

## About

Swerve base initally generated from the CTRE swerve drive generator and made with the Mk5i L2 configs. CommandSwerveDriveTrain remained mostly the same except for the addition of the limelight vision updates. All other subsystems are custom and some make use of the Yet Another Mechanism System. The Main branch is caught up on all the functional code and the Iowa-Regional-2026 branch has the full git history and the code we competed with at the Iowa Regionals with the changes that we made at the event due to breaking mechanisms.
